* streamchat go example

#+begin_src shell
cargo run -- <api-key> <api-secret> [user-id]

# or set STREAM_API_KEY, STREAM_API_SECRET, and optionally STREAM_USER_ID.
cargo run -- abc a  2


# you can pass an optional expiration in seconds. For a 1-hour token:

cargo run -- <api-key> <api-secret> john 3600

or set STREAM_TOKEN_EXPIRATION_SECS=3600.

Verified with cargo check, which succeeds.
#+end_src
